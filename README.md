## 📋 简介

这是一个基于A股游资陈小群（陈宴群）公开言论、交易记录、媒体报道深度蒸馏出的**短线交易思维框架 Skill**。

不是复制这个人，是提炼他的**认知操作系统**——

- 如何判断情绪周期（启动→发酵→高潮→退潮→冰点）
- 如何识别龙头（能带动板块起涨的才叫龙头）
- 如何做买卖决策（弱转强、分歧转一致、核按钮低吸）
- 如何用极致纪律控制回撤（止损无条件执行）

⚠️ **本 Skill 仅用于学习研究、模拟盘参考、思维训练，不构成任何投资建议。**

## 🧠 核心提炼

### 5个心智模型

| 模型 | 一句话 |
|------|--------|
| **情绪周期钟摆** | 市场就是贪婪和恐惧的循环——启动、发酵、高潮、退潮、冰点，周而复始 |
| **龙头辨识度** | 龙头不是猜出来的，是走出来的。能带动板块起涨的才叫龙头 |
| **合力优先** | 散户最大的误区是以为游资抱团有钱就为所欲为——没有市场合力，你砸下去散户接不住的 |
| **弱转强识别** | 机会藏在分歧转一致的瞬间——该弱不弱视为强，该强不强视为弱 |
| **逻辑驱动与证伪** | 预判只是假设，市场不认立刻纠错——不YY |

### 10条决策启发式

1. 只做主线龙头，不碰杂毛
2. 龙头首阴，次日反包 = 行情继续
3. 核按钮日低吸，次日弱转强加仓
4. 退潮期空仓，冰点期看反弹
5. 分歧买入，一致卖出
6. 换手10%-30%健康，拒绝缩量加速和死亡换手（>60%）
7. 入2日未达预期 = 平仓走人
8. 龙头在水下超30分钟（10点上不0轴）= 减仓或出局
9. 分仓控制回撤，单票≤总资金30%
10. 预判是假设，不YY
